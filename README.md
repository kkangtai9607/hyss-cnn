# HySS-CNN

This repository is associated with the manuscript:

**HySS-CNN: A Factorized Spectral–Spatial CNN with Dual-Statistics Pooling for Hyperspectral Image Classification**

The source code, configuration files, and instructions for reproducing the experiments will be released immediately upon acceptance/publication of the paper.

At the current peer-review stage, the repository is used as a placeholder for the official implementation. The final released version will include the training/testing scripts, dataset preparation instructions, model configuration files, and reproducibility details.
